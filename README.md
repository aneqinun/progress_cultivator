# Progress Knight Quest

A continuation of Progress Knight 2.0. Contains 5 prestige layers with various unique mechanics.

## Run locally

No build step is required. This project is plain HTML/CSS/JS.

### Option 1: Open directly

1. Clone this repository.
2. Open `index.html` in your browser.

### Option 2: Run a local web server (recommended)

From the project root, run:

```bash
python -m http.server 8000
```

Then open:

`http://localhost:8000`
